## Описание
Графическое приложение для поиска пользователей на GitHub с возможностью добавления их в избранное. Данные об избранных пользователях сохраняются локально в файл `favorites.json`. Используется публичный GitHub API.

## Требования
- Python 3.8+
- Библиотеки: `requests`, `tkinter` (встроен)

## Установка и запуск
1. Клонируйте репозиторий:
   ```
   git clone https://github.com/yourname/github-user-finder.git
   ```
2. Перейдите в папку проекта.
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
4. Запустите приложение:
   ```
   python main.py
   ```

## Использование API
Приложение использует GitHub REST API: `GET https://api.github.com/users/{username}`.
Для анонимного использования действует лимит: 60 запросов в час.

## Примеры использования
1. Введите имя пользователя (например, `torvalds`) в поле поиска и нажмите «Поиск».
2. Выберите пользователя из списка.
3. Нажмите «Добавить в избранное».
4. Избранные пользователи сохраняются в `favorites.json`.
